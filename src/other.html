The _vdk.isModeIframe() method

When webusers are in a conversation, the website is dsplayed inside an iframe. Use the _vdk.isModeIframe to find out if your website in an iframe: 


_vdk.isModeIframe();

Anatomy

The _vdk.isModeIframe method returns a boolean:

  • true (bool) if your website is inside an iframe.
  • false (bool) if your website is not inside an iframe.

Example


          if(_vdk.isModeIframe() == true){
            // do something
          } else {
            // do something else
          }

You can use this function to hide some components when your website is inside an iframe (call button for example).
